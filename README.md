# video-game-store-database

A relational database management system designed to simulate the operations of a retail video game store. This project demonstrates relational database design, SQL programming, and business reporting using Microsoft SQL Server.

## Overview

This project was developed as part of a database systems course at Arizona State University.

The objective was to design and implement a fully functional relational database capable of managing customers, publishers, video games, orders, reviews, and promotional discounts while maintaining data integrity and supporting business reporting.

The project demonstrates database normalization, SQL programming, stored procedures, triggers, views, and transactional data management using Microsoft SQL Server.

## Technologies

- Microsoft SQL Server
- T-SQL
- SQL Server Management Studio

## Features

- Designed a normalized relational database schema
- Implemented primary and foreign key relationships
- Enforced referential integrity through constraints
- Populated the database with sample transactional data
- Created SQL views for business reporting
- Developed stored procedures for sales analysis
- Implemented triggers to automatically audit database changes
- Used cursors to perform row-by-row processing
